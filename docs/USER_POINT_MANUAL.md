## 포인트 개요

### 포인트란?
포인트는 캠페인 참여 및 리뷰 작성 등의 활동을 통해 적립되는 혜택입니다. 적립된 포인트는 현금으로 출금할 수 있습니다.

### 포인트 적립 방법
- 캠페인 참여 및 완료
- 리뷰 작성 및 승인
- 이벤트 참여

### 포인트 유효기간
- 적립된 포인트는 **60일간** 유효합니다.
- 유효기간이 지나면 자동으로 소멸됩니다.
- 만료 30일 전부터 만료 예정 포인트를 확인할 수 있습니다.

## 포인트 조회

### 접근 경로
```
마이페이지 > 포인트 내역
또는
/mypage/point
```

### 포인트 정보 확인
포인트 내역 페이지에서 다음 정보를 확인할 수 있습니다:

1. **총 적립 포인트**
   - 현재까지 적립된 전체 포인트 (만료되지 않은 포인트)

2. **사용한 포인트**
   - 출금 또는 사용으로 차감된 포인트

3. **잔여 포인트**
   - 현재 사용 가능한 포인트 (총 적립 - 사용)

4. **만료 예정 포인트**
   - 30일 이내에 만료될 포인트

### 포인트 내역 조회
- 적립 내역: 캠페인 완료, 리뷰 승인 등
- 사용 내역: 출금 승인
- 각 내역의 상세 정보 (날짜, 금액, 사유, 만료일)

## 계좌 정보 관리

### 계좌 정보 등록

#### 접근 경로
```
마이페이지 > 출금 계좌 정보
또는
/mypage/account
```

#### 등록 절차
1. 마이페이지에서 "출금 계좌 정보" 메뉴 클릭
2. 계좌 정보 입력
   - **은행명**: 예) 국민은행, 신한은행
   - **계좌번호**: '-' 없이 숫자만 입력
   - **예금주**: 본인 명의 확인
3. "저장" 버튼 클릭

#### 주의사항
- 반드시 **본인 명의**의 계좌를 등록해주세요.
- 잘못된 계좌 정보로 인한 문제는 책임지지 않습니다.
- 계좌 정보는 언제든지 수정할 수 있습니다.

### 계좌 정보 수정
1. "출금 계좌 정보" 페이지 접속
2. 기존 정보 확인 후 수정
3. "저장" 버튼 클릭

### 자동 입력 기능
등록된 계좌 정보는 출금 요청 시 자동으로 입력됩니다.

## 출금 요청

### 출금 조건
- **최소 출금 포인트**: 10,000P
- **출금 가능 포인트**: 잔여 포인트 (대기 중인 출금 제외)
- **본인 명의 계좌** 필수

### 출금 요청 절차

#### 1단계: 출금 요청 페이지 접속
```
마이페이지 > 포인트 내역 > 출금 요청
또는
/mypage/withdrawal-request
```

#### 2단계: 출금 정보 입력
1. **출금 포인트**
   - 출금할 포인트 입력 (최소 10,000P)
   - 1,000P 단위로 입력 권장

2. **계좌 정보**
   - 사전 등록된 계좌 정보가 자동으로 입력됩니다.
   - 미등록 시 직접 입력하거나 "계좌 정보 등록" 링크를 클릭하여 등록할 수 있습니다.

3. **확인 및 제출**
   - 입력 정보 확인
   - "출금 요청" 버튼 클릭

#### 3단계: 요청 상태 확인
출금 요청 후 다음 상태로 진행됩니다:

1. **대기** (노란색)
   - 관리자 검토 대기 중
   - 이 단계에서 취소 가능

2. **승인** (파란색)
   - 관리자가 승인 완료
   - 포인트 차감 완료
   - 출금 진행 중

3. **완료** (초록색)
   - 계좌로 입금 완료

4. **거절** (빨간색)
   - 승인 거절
   - 거절 사유 확인 가능

### 출금 요청 내역 확인
```
마이페이지 > 출금 요청
```
- 모든 출금 요청 내역을 확인할 수 있습니다.
- 각 요청의 상태, 금액, 계좌 정보, 처리 일시를 확인할 수 있습니다.

---

## 출금 요청 취소

### 취소 가능 조건
- **대기** 상태인 경우에만 취소 가능
- 승인 이후에는 취소 불가

### 취소 절차
1. 마이페이지 > 출금 요청 페이지 접속
2. 취소하려는 요청 찾기
3. "취소" 버튼 클릭
4. 확인 메시지에서 "확인" 클릭

### 취소 후
- 요청이 즉시 취소됩니다.
- 포인트는 차감되지 않습니다.
- 필요시 다시 출금 요청할 수 있습니다.

## FAQ

### Q1. 출금은 얼마나 걸리나요?
**A.** 출금 요청 후 영업일 기준 3~5일 소요됩니다.
- 대기 → 승인: 1~2일
- 승인 → 완료: 2~3일

### Q2. 최소 출금 금액이 있나요?
**A.** 네, 최소 10,000P부터 출금 가능합니다.

### Q3. 출금 수수료가 있나요?
**A.** 현재 출금 수수료는 없습니다.

### Q4. 타인 명의 계좌로 출금 가능한가요?
**A.** 아니요, 반드시 본인 명의 계좌로만 출금 가능합니다.

### Q5. 출금이 거절되면 어떻게 되나요?
**A.** 거절된 경우:
- 포인트는 차감되지 않습니다.
- 거절 사유를 확인할 수 있습니다.
- 사유 확인 후 다시 출금 요청할 수 있습니다.

### Q6. 포인트 만료 전에 알림을 받을 수 있나요?
**A.** 포인트 내역 페이지에서 만료 예정 포인트를 확인할 수 있습니다. (30일 이내 만료 예정)

### Q7. 계좌 정보를 잘못 입력했어요.
**A.**
- **대기 상태**: 요청을 취소하고 계좌 정보 수정 후 다시 요청
- **승인 이후**: 고객센터로 즉시 문의

### Q8. 여러 건의 출금 요청을 동시에 할 수 있나요?
**A.** 네, 가능합니다. 단, 잔여 포인트 범위 내에서만 요청 가능합니다.

### Q9. 출금 요청 후 포인트가 차감되나요?
**A.**
- **대기 상태**: 포인트 차감 안 됨 (출금 가능 포인트에서만 제외)
- **승인 시**: 포인트 차감

### Q10. 문의는 어디로 하나요?
