# 포인트 시스템 요구사항

## 기본 요구사항

* 캠페인 등록시 포인트점수를 설정할수있음
* 사용자들은 해당 캠페인을 수행하고 캠페인이 마감되면 포인트가 적립됨
* 관리자는 사용자들의 포인트를 확인하고 관리할 수 있어야함
* 포인트는 현금으로 수기로 전환하고 관리자페이지에서 소진처리를 해야함

## 추가 요구사항 (출금 시스템)

* 사용자가 포인트 출금을 요청할 수 있음 (최소 10,000 포인트)
* 출금 요청 시 은행 정보(은행명, 계좌번호, 예금주) 입력
* 출금 요청 시 관리자에게 알림 전송
* 관리자가 출금 요청을 승인/거부할 수 있음
* 승인 시 자동으로 포인트 차감
* 거부 시 거부 사유 입력 필수
